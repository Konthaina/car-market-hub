# Car Market Hub - Frontend

React + Vite frontend for the Car Market Hub application.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- **Login Page**: Beautiful, modern login interface with form validation
- **Authentication**: Integrated with Laravel Sanctum for secure authentication
- **State Management**: Zustand for auth state management
- **Routing**: React Router for navigation
- **Styling**: Tailwind CSS for modern, responsive design

## API Configuration

The frontend is configured to proxy API requests to `http://127.0.0.1:8000` (Laravel backend). Make sure your backend is running on that port.
